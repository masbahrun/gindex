# GIndex

GIndex is a Python script to publish url in Google Index v3.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install -r requirements.txt
```

## Usage

```bash
python gindex.py json_key_file.json
```
